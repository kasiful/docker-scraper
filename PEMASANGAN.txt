# SERVER SCRAPER BIGDATA BPS #

ini adalah aplikasi scraper menggunakan docker yang terdiri dari:
- ubuntu yang terinstall airflow, python, chrome, dan php
- database MariaDB
- phpmyadmin

cara install:

1. pasang semua docker dengan mengetik shell 
=> docker-compose up -d
jika gagal, ulangi terus hingga selesai download

2. masuk ke bash ubuntu docker
=> docker exec -it ubuntu_server_1 bash

3. aktifkan airflow secara manual, utk instalasi skr command aktivasi tinggal run activate.sh
=> sh /home/settings/activate.sh

jika semua sudah jalan, maka bisa buka browser, url:
home       => http://0.0.0.0:8000
airflow    => http://0.0.0.0:8080
phpmyadmin => http://0.0.0.0:8081
